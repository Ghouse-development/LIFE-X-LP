# LIFE X FC LP - 表記・注釈ガイドライン

このドキュメントは、LIFE X フランチャイズLPにおける性能値・金融・価格に関する表記ルールを定めます。

---

## 1. 性能値の表記ルール

### 1.1 断熱性能（UA値）

**表記例**:
> UA値 0.46以下（HEAT20 G2相当）

**必須注釈**:
```
※数値は一例であり、仕様・地域・設計条件により変動します。
※HEAT20 G2相当は6地域（東京・大阪等）における基準値です。
※詳細は個別面談時に設計図書をもとにご説明いたします。
```

**根拠資料**:
- 一般社団法人 20年先を見据えた日本の高断熱住宅研究会「HEAT20設計ガイドブック」
- URL: https://www.heat20.jp/

---

### 1.2 気密性能（C値）

**表記例**:
> C値 0.5以下（実測値）

**必須注釈**:
```
※C値は全棟測定を実施し、実測値に基づく表記です。
※気密測定は第三者機関による測定を推奨しています。
※地域・施工精度により変動する場合があります。
```

**根拠資料**:
- 一般社団法人 日本建材・住宅設備産業協会「住宅の省エネルギー基準」
- 気密測定技術基準（JIS A 2201準拠）

---

### 1.3 耐震性能

**表記例**:
> 耐震等級3（許容応力度計算）

**必須注釈**:
```
※耐震等級3は建築基準法の1.5倍の耐震性を意味します。
※許容応力度計算に基づく構造設計を実施しています。
※詳細は構造計算書をもとに個別面談時にご説明いたします。
```

**根拠資料**:
- 国土交通省「住宅性能表示制度」耐震等級基準
- 建築基準法施行令 第82条（許容応力度）

---

### 1.4 保証期間

**表記例**:
> 初期保証20年（構造・防水）

**必須注釈**:
```
※保証内容は契約書に基づきます。詳細は契約前面談でご確認ください。
※地盤保証は第三者機関による20年保証を別途付帯します。
※定期点検の受診が保証継続の条件となります。
```

---

## 2. 金融・収益モデルの表記ルール

### 2.1 収益試算

**表記例**:
> 1棟あたり粗利 500〜800万円

**必須注釈**:
```
※以上は参考値です。エリア・施策・坪単価により変動します。
※確定値は契約前面談で開示します。
※初期投資・販促費・人件費等は別途必要です。
```

**根拠資料**:
- 過去3年間の加盟店実績データ（匿名化済み）
- 業界平均坪単価データ（住宅産業新聞調べ）

---

### 2.2 初期投資

**表記例**:
> 初期投資 300万円〜

**必須注釈**:
```
※上記は最小構成（社長1名＋営業1名）の場合の目安です。
※加盟金・研修費・初期広告費・システム導入費を含みます。
※人件費・事務所家賃・什器備品等は別途必要です。
※詳細は契約前面談で見積書を提示いたします。
```

---

### 2.3 損益分岐点

**表記例**:
> 損益分岐点 4〜6棟（初年度）

**必須注釈**:
```
※最小構成・初期投資300万円・坪単価60万円を前提とした試算です。
※販促費・人件費・間接費の水準により変動します。
※確定値は契約前面談で収支シミュレーションをご提示します。
```

## 3. 価格表記ルール

### 3.1 坪単価

**表記例**:
> 坪単価 55〜70万円想定

**必須注釈**:
```
※坪単価は本体工事費のみを含み、付帯工事・外構・諸費用は含みません。
※仕様変更・オプション追加により変動します。
※地域・施工条件により別途調整が発生する場合があります。
```

※ロイヤリティの詳細は契約前面談で開示いたします。
